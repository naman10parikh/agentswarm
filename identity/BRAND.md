# Hive — Brand

## Name Rationale

**Hive** — a hive coordinates a superorganism: one queen, many workers, fractal sub-colonies. The word perfectly captures CEO→VP→worker dispatch and recursive sub-swarm spawning. One word, biological, immediately understood.

## Tagline

> One YAML. A swarm of experts. Zero idle panes.

## Colors

Warm black (#141312) background · Honey amber (#d97706) for active swarms · Zinc-500 for waiting VPs · Green-400 for completed waves. Poppins for headings, monospace for YAML configs.

## Landing Copy

Hive reads a YAML and becomes a CEO. Define your roles, set the dependencies, and watch a swarm of Claude Code agents execute in wave order — each VP starting only when the one it depends on is done. Fractal by design: any VP can spawn its own Hive.
